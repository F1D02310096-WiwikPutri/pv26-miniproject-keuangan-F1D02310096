# Aplikasi Keuangan Sederhana

## Deskripsi

Aplikasi Keuangan Sederhana adalah aplikasi berbasis desktop yang dibuat menggunakan PySide6 dan SQLite. Aplikasi ini digunakan untuk mencatat dan mengelola transaksi keuangan berupa pemasukan dan pengeluaran secara terstruktur.

## Fitur utama yang tersedia meliputi:

- Menambahkan data transaksi
- Menampilkan data transaksi
- Mengedit data transaksi
- Menghapus data transaksi
  Data yang dimasukkan akan tersimpan secara permanen di database SQLite sehingga tetap tersedia meskipun aplikasi ditutup.

## Cara Menjalankan

1. Pastikan Python sudah terinstall di komputer.
2. Install dependency PySide6:
   pip install PySide6
3. Jalankan file utama:
   python main.py

## Teknologi yang Digunakan

- Python
- PySide6 (GUI Framework)
- SQLite (Database)
- QSS (Styling)
